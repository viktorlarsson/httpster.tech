## Getting started

Install bun 

```bash
curl -fsSL https://bun.sh/install | bash
```

Run installation
```bash
bun install
```

Run dev
```bash
bun run dev
```

Run test
```bash
bun run test
```

Run lint
```bash
bun run lint
```
